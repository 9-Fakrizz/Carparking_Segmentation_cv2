# 🚗 Smart Parking Detection System with Telegram Notification

ระบบตรวจจับที่จอดรถว่าง/ไม่ว่าง แบบ Real-time ด้วยกล้อง Webcam/CCTV และ Python OpenCV พร้อมแจ้งเตือนสถานะของทุกช่องผ่าน Telegram ทันทีที่มีการเปลี่ยนแปลง

## 📋 Requirements (สิ่งที่ต้องมี)

* Python 3.x
* Webcam หรือ CCTV (ที่เชื่อมต่อกับคอมพิวเตอร์)
* Internet (สำหรับการส่งข้อความเข้า Telegram)

## 🛠️ Installation (การติดตั้ง)

1. **ติดตั้ง Python Libraries**
เปิด Terminal หรือ Command Prompt แล้วรันคำสั่งต่อไปนี้:
```bash
pip install opencv-python numpy requests

```


2. **เตรียม Telegram Bot**
* ทักแชทไปหา **BotFather** ใน Telegram เพื่อสร้างบอทใหม่และรับ `BOT_TOKEN`
* หา `CHAT_ID` ของตัวคุณเอง (หรือกลุ่ม) โดยใช้บอทช่วย เช่น **@userinfobot**
* นำค่าทั้งสองมาใส่ในโค้ดส่วน `TELEGRAM SETTINGS`



## ⚙️ Configuration (การตั้งค่า)

แก้ไขตัวแปรในไฟล์ `main.py` (หรือชื่อไฟล์ของคุณ) ดังนี้:

```python
# ใส่ Token ที่ได้จาก BotFather
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE" 

# ใส่ Chat ID ของคุณ
CHAT_ID   = "YOUR_CHAT_ID_HERE"

# เลือกกล้อง (0 = กล้องเว็บแคมในเครื่อง, 1 หรือ URL = กล้องตัวอื่น)
cap = cv2.VideoCapture(0) 

```

> **วิธีตั้งจุดจอดรถ (Coordinates):**
> หากมุมกล้องเปลี่ยน คุณต้องแก้ค่าพิกัดในตัวแปร `parking_areas` ให้ตรงกับช่องจอดจริง โดยค่าจะเป็น `[x, y]` ของมุมทั้ง 4 มุมในแต่ละช่อง

## 🚀 How to Run (วิธีใช้งาน)

รันคำสั่งใน Terminal:

```bash
python parking_detect.py

```

* กดปุ่ม **`q`** บนคีย์บอร์ดเพื่อปิดโปรแกรม

## 🧠 How It Works (หลักการทำงาน)

ระบบทำงานเป็นขั้นตอนดังนี้:

1. **Image Acquisition**: โปรแกรมรับภาพจากกล้องทีละเฟรม
2. **Preprocessing (HSV Color Space)**:
* แปลงภาพเป็นโหมดสี HSV
* ใช้ `cv2.inRange` เพื่อแยกสีของ "วัตถุ" (รถ) ออกจาก "พื้นถนน" (สมมติว่าพื้นเป็นสีเทา/ดำ และรถมีสีสัน หรือเงาที่ต่างออกไป)


3. **Noise Reduction**: ใช้ `MedianBlur` และ `Morphology` (Open/Close) เพื่อลบจุดรบกวนเล็กๆ ออก ทำให้ภาพ Mask สะอาดขึ้น
4. **ROI Monitoring (Region of Interest)**:
* โปรแกรมจะวาดพื้นที่สี่เหลี่ยม 4 ช่องตามพิกัดที่ตั้งไว้ (`parking_areas`)
* นับจำนวนเม็ดพิกเซลสีขาว (White Pixels) ในแต่ละช่อง


5. **Occupancy Logic**:
* ถ้าจำนวนพิกเซลสีขาว > 2000 (ค่า Threshold) → ถือว่า **"มีรถจอด (Occupied)"**
* ถ้าน้อยกว่า → ถือว่า **"ว่าง (Free)"**


6. **Stability Filter (กันไฟกระพริบ)**:
* ระบบจะไม่เปลี่ยนสถานะทันทีที่ค่าเปลี่ยนแค่เสี้ยววินาที
* ต้องมีค่าสถานะเหมือนเดิมติดต่อกัน `8 เฟรม` (`STABLE_FRAMES`) ถึงจะยืนยันการเปลี่ยนสถานะ (ป้องกันแสงวูบวาบหรือคนเดินผ่านเร็วๆ)


7. **Notification**:
* เมื่อสถานะช่องใดช่องหนึ่งเปลี่ยนและยืนยันแล้ว ("เสถียรแล้ว")
* ระบบจะรวบรวมสถานะของ **ทั้ง 4 ช่อง**
* ส่งข้อความพร้อมรูปภาพล่าสุดไปยัง Telegram ทันที



---

### 💡 Tips

* **การตั้งค่าสี (Color Threshold):** หากแสงเปลี่ยน หรือพื้นเปลี่ยนสี อาจต้องปรับค่า `lower` และ `upper` ในบรรทัด `cv2.inRange` ให้เหมาะสมกับหน้างานจริง
* **ปรับความไว:** แก้ไขค่า `white_pixels > 2000` ถ้าต้องการให้ตรวจจับได้ไวขึ้น (ลดตัวเลข) หรือลดความผิดพลาดจากเงา (เพิ่มตัวเลข)
